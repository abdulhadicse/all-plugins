## Cat Fact Widget Plugin
In the plugin where you'll have to add a new dashboard widget and display 5 Cat facts (https://alexwohlbruck.github.io/cat-facts/docs/), Use Transient api to store cat facts for 24 hours.

## Technologies
* PHP: > 7.2
* Wordpress: > 5.2
* Plugin Version: 1.0.0

## Setup
```
To run this project, just download & install & active it.
```
